########## gemini v1.0.0 ##########

gemini致力于做更简洁，更通用的后台管理系统

父项目：主要管理pom文件中jar包的版本。

########## gemini-common ##########

公共模块

########## gemini-core ##########

核心模块

**技术选型** 
-   核心框架        -   Spring boot
-   持久层框架      -   Mybatis
-   权限框架        -   Apache Shiro
-   数据库连接池    -   Alibaba Druid
-   日志框架        -   LogBack
-   缓存框架        -   Redis
-   数据库          -   Mysql


########## gemini-layui ##########

layui前端模块

**技术选型** 
-   核心框架        -   layui
-   表格            -   layui-table
-   树形表格        -   layui-treetable-lay
-   树              -   layui-tree
-   权限树          -   layui-authTree-v1.1
-   下拉多选        -   layui-formSelects-v4
-   图形报表        -   echarts
